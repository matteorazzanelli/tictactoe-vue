<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo

```sh
git clone git@github.com:matteorazzanelli/tictactoe-vue.git
```

2. Project setup
```
npm install
```

3. Compiles and hot-reloads for development
```
npm run serve
```

4. Go to http://localhost:8080

<!-- USAGE -->
## Usage

- Type two feasbile names (i.e. more than 2 characters)
- Play the game
